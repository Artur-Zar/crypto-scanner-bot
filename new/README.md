# 🔍 HR Vacancy Tech Stack Analyzer

Простой и эффективный инструмент на Python для автоматического извлечения технологий из описаний вакансий.

## 🚀 Функции
- **Локальный анализ:** Работает без интернета и внешних API.
- **Regex Matching:** Точный поиск технологий с учетом границ слов.
- **Clean Code:** Написано с использованием Type Hinting и Logging.

## 🛠 Как запустить
1. Склонируйте репозиторий.
2. Добавьте текст вакансии в `vacancy.txt`.
3. Запустите: `python hr_helper.py`.
