# 📈 Crypto Volatility Scanner Bot

Telegram-бот для автоматического поиска самых волатильных торговых пар на фьючерсном рынке крупнейших криптобирж.

## 🚀 Основные возможности
- **Мультибиржа:** Сканирует Binance, Bybit, OKX, Gate.io и Bitget одновременно.
- **Фьючерсный рынок:** Работает с бессрочными контрактами (Swap).
- **Фильтрация:** Игнорирует "мусорные" монеты с низким объемом торгов.
- **Интеграция с TradingView:** Прямые ссылки на графики в отчете.

## 🛠 Технологии
- Python 3.x
- ccxt (API криптобирж)
- pyTelegramBotAPI
- python-dotenv

## 📦 Как запустить проект у себя

1. **Клонируйте репозиторий:**
   `git clone https://github.com/Artur-Zar/crypto-scanner-bot.git`

2. **Установите зависимости:**
   `pip install -r requirements.txt`

3. **Настройте переменные окружения:**
   Создайте файл `.env` и добавьте туда ваш токен:
   `TELEGRAM_TOKEN=ваш_токен_от_BotFather`

4. **Запустите бота:**
   `python main.py`

## 📝 Команды
- /start — Приветствие и инструкция.
- /scan — Запуск сканирования и получение ТОП-10 волатильных пар.
